# TimeLens — Product Vision

## Problem Statement

Knowledge workers lose approximately 40% of their productive potential not because they lack time, but because they lack awareness of how their mental states shape their experience of time.

Current tools measure *clock time*. No tool measures *cognitive time*.

## The Core Insight

Neuroscience shows that time perception is profoundly subjective:
- In **flow state**, hours feel like minutes
- In **boredom**, minutes feel like hours  
- **Fear** stretches time; **joy** compresses it

If we can help people understand their cognitive time patterns, we can help them schedule, focus, and recover with dramatically higher precision than any calendar or timer app.

## Unique Value Proposition

> "TimeLens doesn't track time. It optimizes how time is experienced."

We quantify the unquantifiable: the gap between clock reality and subjective experience. Then we use that data to make you more effective.

## Target Users

### Primary: The Deep Work Professional
- Developers, writers, designers
- Need uninterrupted focus blocks
- Frustrated by how fast "good" days feel vs. "bad" days

### Secondary: The Academic
- Students managing study vs. distraction
- Need to understand when their cognitive peaks occur

### Tertiary: The Self-Optimizer
- Biohackers, productivity enthusiasts
- Obsessed with data about their own performance

## Feature Roadmap

### Phase 1 — MVP (Current)
- Perception measurement engine (hidden clock + blind estimate)
- Session tracking with measured ratio / accuracy / direction
- Honest analytics (accuracy, compression rate, per-activity/state breakdowns)
- Insights (free rule-based; Pro LLM provider scaffolded + gated)
- JWT auth, freemium plan field

### Phase 2 — Intelligence
- Ship the Pro LLM insight provider + billing
- Reproduction & production estimation tasks (other chronoception paradigms)
- Personalized perception forecasts from the user's own history
- Calendar integration (Google, Outlook)
- Weekly email digest

### Phase 3 — Ecosystem
- Team dashboards for engineering managers
- Wearable integration (Oura, Whoop, Apple Watch)
- Slack/Notion integrations
- API for third-party developers

### Phase 4 — Platform
- TimeLens SDK
- B2B licensing for enterprises
- Research partnerships with cognitive science labs

## Success Metrics

- **Activation**: % of registered users who complete 3+ sessions in week 1
- **Retention**: 30-day and 90-day active user rate
- **Engagement**: Sessions per active user per week
- **Outcome**: User-reported productivity improvement (NPS + survey)
